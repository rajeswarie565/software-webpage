# Lumina Digital - Landing Page

A responsive, SEO-optimized landing page for a software/digital services company.

## Features

- ✅ Clean, modern, professional light theme
- ✅ Fully responsive design (desktop, tablet, mobile)
- ✅ Cross-platform compatible (Android & Windows)
- ✅ SEO-optimized with proper HTML structure
- ✅ Semantic HTML5 tags
- ✅ Proper heading hierarchy (H1, H2, H3)
- ✅ Meta tags for SEO and social sharing
- ✅ Accessible with ARIA labels
- ✅ Smooth scrolling and animations
- ✅ Mobile-friendly touch targets (44px minimum)

## Files

- `index.html` - Main HTML structure with SEO elements
- `styles.css` - Responsive CSS with light theme
- `script.js` - Interactive features (smooth scroll, mobile menu)

## How to View

1. Open `index.html` in any modern web browser
2. The page is fully self-contained (no build process required)

## Taking Screenshots for Submission

To capture screenshots for your submission:

### Desktop View
1. Open `index.html` in your browser
2. Set browser window to 1920x1080 or similar desktop resolution
3. Take a screenshot of the full page (you may need to scroll and capture multiple sections)

### Mobile View
1. Open browser DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M or Cmd+Shift+M)
3. Select a mobile device (e.g., iPhone 12, Samsung Galaxy)
4. Take screenshots at different scroll positions:
   - Hero section
   - Services section
   - Contact form

### Recommended Screenshots
- Desktop full page (or hero + services)
- Mobile hero section
- Mobile contact form
